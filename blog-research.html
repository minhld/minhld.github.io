Wheelchair Project

  • December 06, 2017

Automatic Wheelchair is a project at the Electrical and Computer Engineering department at Utah State University.

.

In this project, I used ROS platform for video streaming from the camera to the third parties, OpenCV for image processing. The back-end server is built with Java Servlet and MongoDB. For web front-end, I used JQuery and Bootstrap libraries for interactions and template.
